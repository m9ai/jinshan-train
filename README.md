# 金山铁路时刻表

上海地区金山铁路班次查询工具，覆盖全线 9 座站点，支持 Chrome / Edge 浏览器扩展与微信小程序双端使用。

## 核心功能

- **智能班次查询** — 选择出发站与到达站，一键查询所有匹配班次。支持直达、大站停、站站停三种车次类型筛选，以及仅看未发车过滤。
- **14 天预排期** — 支持查看未来 14 天内的列车时刻，自动识别平日方案与周末/节假日方案。
- **班次收藏** — 收藏日常通勤班次，在「我的收藏」中快速查看始发到站时间。
- **站点详情** — 查看沿线 9 座站点的地址、营业时间、地铁换乘、公交换乘信息，支持一键导航。
- **票价查询** — 内置全线票价矩阵表，上海南至金山卫票价 3-10 元不等。
- **深色模式** — 支持深色/浅色/跟随系统三种主题切换。
